# Từ vựng và Ngữ pháp Tiếng Anh

Từ vựng và ngữ pháp là hai thành phần cơ bản và thiết yếu trong việc học tiếng Anh. Trong kỳ thi vào lớp 6, phần này chiếm một tỷ trọng lớn trong đề thi và đánh giá khả năng sử dụng ngôn ngữ của học sinh.

## 1. Tầm quan trọng của từ vựng và ngữ pháp

```mermaid
mindmap
  root((Từ vựng & Ngữ pháp))
    ((Nền tảng giao tiếp))
      (Hiểu được thông điệp)
      (Truyền đạt ý tưởng)
      (Xây dựng câu đúng)
    ((Phát triển kỹ năng))
      (Hỗ trợ kỹ năng nghe)
      (Nâng cao khả năng đọc)
      (Cải thiện kỹ năng viết)
      (Tăng sự tự tin khi nói)
    ((Thành công trong học tập))
      (Làm bài thi tốt)
      (Hiểu sách giáo khoa)
      (Tham gia hoạt động)
    ((Ứng dụng thực tế))
      (Xem phim, nghe nhạc)
      (Sử dụng internet)
      (Giao tiếp với người nước ngoài)
```

        +Môi trường
        +Động vật & Thực vật

        +Mua sắm & Tiền bạc
    }

        +Thành viên gia đình
        +Mối quan hệ
        +Hoạt động gia đình
    }
    
    class TrườngHọc {
        +Đồ dùng học tập

    ChủĐềTừVựng <|-- GiaĐình

## 3. Các điểm ngữ pháp cơ bản

### 3.1. Thì (Tenses)

```mermaid
graph TD
    A[Thì trong tiếng Anh] --> B[Hiện tại]
    A --> C[Quá khứ]
    A --> D[Tương lai]
    
    B --> B1[Hiện tại đơn]
    B --> B2[Hiện tại tiếp diễn]
    B --> B3[Hiện tại hoàn thành]
    
    C --> C1[Quá khứ đơn]
    C --> C2[Quá khứ tiếp diễn]
    
    D --> D1[Tương lai đơn]
    D --> D2[Tương lai gần]
    
    B1 --> B1a["S + V(s/es)"]
    B1 --> B1b["Dấu hiệu: always, usually, often..."]
    
    B2 --> B2a["S + am/is/are + V-ing"]
    B2 --> B2b["Dấu hiệu: now, at the moment..."]
    
    C1 --> C1a["S + V-ed/V2"]
    C1 --> C1b["Dấu hiệu: yesterday, last week..."]
    
    D1 --> D1a["S + will + V"]
    D1 --> D1b["Dấu hiệu: tomorrow, next week..."]
    
    D2 --> D2a["S + am/is/are + going to + V"]
    D2 --> D2b["Dấu hiệu: kế hoạch đã định trước"]

    style B1a,B1b,B2a,B2b,C1a,C1b,D1a,D1b,D2a,D2b fill:#d5f9e5,stroke:#333,stroke-width:1px
```

### 3.2. Từ loại (Parts of speech)

```mermaid
mindmap
  root((Từ loại))
    ((Danh từ - Nouns))
      (Danh từ chung: book, dog, teacher)
      (Danh từ riêng: London, Mary, Monday)
      (Danh từ đếm được: apple, student)
      (Danh từ không đếm được: water, rice)
    ((Động từ - Verbs))
      (Động từ thường: eat, study, play)
      (Trợ động từ: be, do, have)
      (Động từ khiếm khuyết: can, must, should)
    ((Tính từ - Adjectives))
      (Mô tả tính chất: beautiful, tall, big)
      (Chỉ định: this, that, these, those)
      (Số lượng: many, few, some)
    ((Trạng từ - Adverbs))
      (Chỉ cách thức: slowly, quickly)
      (Chỉ thời gian: now, yesterday)
      (Chỉ nơi chốn: here, there)
    ((Giới từ - Prepositions))
      (Chỉ vị trí: in, on, at)
      (Chỉ thời gian: before, after)
      (Chỉ hướng: to, from, towards)
    ((Đại từ - Pronouns))
      (Nhân xưng: I, you, he, she)
      (Sở hữu: my, your, his, her)
      (Phản thân: myself, yourself)
```

## 4. Các dạng bài tập thường gặp

### 4.1. Bài tập về từ vựng

```mermaid
classDiagram
    class BàiTậpTừVựng {
        +Điền từ vào chỗ trống
        +Chọn từ đúng
        +Nối từ với nghĩa
        +Tìm từ trái nghĩa
        +Tìm từ đồng nghĩa
        +Sắp xếp chữ cái
    }
    
    class ĐiềnTừ {
        +Dựa vào ngữ cảnh
        +Từ cho sẵn
        +Không có từ gợi ý
    }
    
    class ChọnTừ {
        +Chọn 1 trong 4 đáp án
        +Chọn từ đúng về nghĩa
        +Chọn từ đúng về ngữ cảnh
    }
    
    class NốiTừ {
        +Nối từ với định nghĩa
        +Nối từ với hình ảnh
        +Nối từ với từ loại
    }
    
    BàiTậpTừVựng <|-- ĐiềnTừ
    BàiTậpTừVựng <|-- ChọnTừ
    BàiTậpTừVựng <|-- NốiTừ
```

## 7. Các lỗi thường gặp và cách khắc phục

```mermaid
flowchart TD
    A[Lỗi thường gặp] --> B{Loại lỗi?}
    B -->|Từ vựng| C[Lỗi từ vựng]
    B -->|Ngữ pháp| D[Lỗi ngữ pháp]
    
    C --> C1[Dùng sai từ]
    C --> C2[Không nhớ từ]
    C --> C3[Phát âm sai]
    
    D --> D1[Sai cấu trúc câu]
    D --> D2[Sai thì động từ]
    D --> D3[Sai cách dùng mạo từ]
    D --> D4[Sai cách dùng giới từ]
    
    C1 --> C1a[Học từ trong ngữ cảnh]
    C2 --> C2a[Ôn tập thường xuyên với flashcards]
    C3 --> C3a[Luyện phát âm và nghe]
    
    D1 --> D1a[Học mẫu câu chuẩn]
    D2 --> D2a[Làm bài tập về thì]
    D3 --> D3a[Học quy tắc dùng mạo từ]
    D4 --> D4a[Học cụm từ có giới từ]
    
    style A fill:#f9d5e5,stroke:#333,stroke-width:2px
    style B fill:#f9e5d5,stroke:#333,stroke-width:2px
    style C,D fill:#d5e5f9,stroke:#333,stroke-width:2px
    style C1,C2,C3,D1,D2,D3,D4 fill:#e5f9d5,stroke:#333,stroke-width:1px
    style C1a,C2a,C3a,D1a,D2a,D3a,D4a fill:#d5f9e5,stroke:#333,stroke-width:1px
```

---

Để làm tốt phần từ vựng và ngữ pháp, học sinh cần học từng bước, luyện tập thường xuyên và áp dụng vào thực tế. Hãy tạo thói quen học từ mới mỗi ngày, làm bài tập ngữ pháp thường xuyên và đừng quên luyện tập tổng hợp qua các dạng bài thi thực tế. Ngoài ra, việc đọc sách, nghe nhạc, xem phim bằng tiếng Anh cũng giúp cải thiện vốn từ vựng và ngữ pháp một cách tự nhiên.
